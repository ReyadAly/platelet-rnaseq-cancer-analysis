# platelet-rnaseq-cancer-analysis
RNA-seq platelet data analysis for cancer type classification using Python

## Dataset
Download the dataset from Kaggle:
    https://www.kaggle.com/datasets/samiraalipour/gene-expression-omnibus-geo-dataset-gse68086

After downloading, place the CSV file in the same folder as the notebook and run:
    df = pd.read_csv("GSE68086_TEP_data_matrix.csv")
